Welcome to Homeworker Helper!

Introduction

Homeworker Helper (HH) is a web-based tool focused on helping you do more with your time! Unlike other time-management apps that simply let you create a schedule with tasks, HH allows you to set specific websites and domains that you want to work on, and, for a set period of time, allows you to only access those websites. Any time you go to a site that you did not specify earlier, HH uses verbal encouragement to persuade you to return to doing your work.
